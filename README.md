# Netflix-Global-Content-Analysis
End to End Netflix Global Content  Data Analysis  Project
        ###########Tool Stack
SQL: Data extraction and initial filtering.
Python (Pandas and numpy): Data cleaning and converting duration to integers.
Python (Matplotlib): Statistical plotting (Histogram, Box Plot, Scatter Plot).
Power BI: Interactive dashboard for final reporting.
       ########## Key Insights
Content Mix: Movies are the dominant content type, making up 69% of the total library.
Global Hubs: The United States and India are the leading countries in content production.
The 95-Minute Rule: Most successful movies on the platform have a runtime between 90 and 110 minutes.
Target Audience: 70% of the content is rated TV-MA or TV-14, focusing on mature and teenage viewers.
Peak Growth: Content volume reached its highest peak in 2019, followed by a decline in 2021.
